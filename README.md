# RAG Learning

A hands-on learning repository for exploring **Retrieval-Augmented Generation (RAG)** using Python and LangChain.

## Topics Covered

* Document Loading with TextLoader
* Working with text documents
* LangChain document loaders
* RAG pipeline fundamentals
* Jupyter Notebook experiments

## Project Structure

```text
RAG-learning/
├── data/
├── notebook/
├── main.py
├── requirements.txt
└── pyproject.toml
```

## Setup

```bash
git clone https://github.com/itsksfit/RAG-learning.git
cd RAG-learning
uv sync
```
